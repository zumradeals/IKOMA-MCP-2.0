# IKOMA MCP 2.0 — BUILD-1 (Squelette)

Ce dépôt contient un squelette minimal aligné sur les ACTES I–IV.

- ACTE I — Carte ontologique du moteur.
- ACTE II — Référentiel d’autorité et d’évidence.
- ACTE III — Modèle des états et transitions.
- ACTE IV — Modèle des ordres, refus et silences.

Aucune logique d’exécution n’est incluse. Ce squelette expose uniquement des contrats,
des frontières et des types.
